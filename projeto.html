Sobre O Projeto Red Core

Objetivo

Consiste em fazer o front-end de uma loja virtual amigável e responsivo na área de tecnologia, em busca de alcançar um nível de página profissional e aprender como criar, na prática, uma página com um bom html e css autoral. Praticando também o trabalho em equipe num projeto de informática.

Público-alvo

Usuários que acessarão nosso site em celulares, computadores, tablets, notebooks, em busca de comprar máquinas e periféricos do básico ao avançado.

Tecnologias Utilizadas

HTML5; CSS3; Chat GPT(para criação da logo e do banner); Git e Github para conectar o projeto entre os desenvolvedores, facilitando bastante a organização dos arquivos da página.

Restrições e requisitos mínimos para usuários

Uma interface para quase todo tipo de usuário, desde usúarios de telas pequenas (~300px) até telas grandes (~2000px). Existe falta de responsividade em telas menores que ~250px, em telas maiores que 2000px, não temos certeza de como ficou a responsividade pela falta de testes.

Dificuldades encontradas

Dificuldade na hora de fazer alguns componentes do site que necessitam de JavaScript, até desistirmos deles. Dificuldade com o design do site no geral, usar cores harmonicas, detalhes de css em certas propriedades, como o detalhe de border-box, e principalmente, na responsividade.

Em alguns casos, tivemos que procurar dicas em fóruns online, como Reddit, para solucionar alguns problemas no site, ou para dúvidas. Um exemplo, para lembrar como alterar o ícone do site na barra do navegador.

Periodicidade de atualização

Uma média de 4-6 atualizações por semana no projeto, que está linkado no git e github para facilitar a cooperação do grupo.

Bootstrap

No nosso site não foi utilizado o bootstrap. Até experimentamos usar em um momento, porém, o site já estava todo formatado no nosso css, com a adição do bootstrap ele mudou tamanho de coisas, fontes, etc. Só preferimos abrir mão dele naquele momento.
