Sobre O Projeto Red Core

Objetivo

Consiste em fazer o front-end de uma loja virtual amigável e responsivo na área de tecnologia, em busca de alcançar um nível de página profissional.

Público-alvo

Usuários que acessarão nosso site em celulares, computadores, tablets, notebooks, em busca de comprar máquinas e periféricos do básico ao avançado.

Tecnologias Utilizadas

HTML5, CSS3, Chat GPT(para criação da logo e do banner).

Restrições e requisitos mínimos para usuários

Uma interface para quase todo tipo de usuário, desde usúarios de telas pequenas (~300px) até telas grandes (~2000px+). Existe falta de responsividade em telas menores que ~250px.

Dificuldades encontradas

Dificuldade na hora de fazer alguns componentes do site que necessitam de JavaScript, até desistirmos deles. Dificuldade com o design do site no geral, usar cores harmonicas, detalhes de css em certas propriedades, como o detalhe de border-box, e principalmente, na responsividade.

Periodicidade de atualização

Uma média de 4-6 atualizações por semana no projeto, que está linkado no git e github para facilitar a cooperação do grupo.

Bootstrap

No nosso site não foi utilizado o bootstrap.
